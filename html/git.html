Git

  1. ls - список папок и файлов в текущей директории, где была выполнена команда.
  2. cd - переход в домашнюю директорию текущего пользователя.
  3. pwd - просмотр вашего текущего местоположения.
  4. mkdir - создать папку.
  5. mv - переместить или переименовать папку/
  6. init - создание нового репозитория в пустой директории или превращение существующей директории в новый репозиторий.
  7. clone URL — клонирование репозитория из удаленного хранилища на локальную машину.
  8. push — выгрузка сделанных изменений в удаленный репозиторий (например, GitHub).
  9. pull — получение изменений из удаленного репозитория на локальную машину.
  10. status — просмотр списка измененных файлов.
  11. branch название_ветки — создание ветки.
  12. checkout название_ветки — переход на ветку.
  13. commit — сохранение изменений в Git.
  14. log — просмотр истории коммитов в репозитории.
  15. add страница.html — добавление в отслеживаемые файлы.
